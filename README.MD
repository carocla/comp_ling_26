# Computational Linguistics: Independent Study (Spring 2026)

This repository contains code, notes, and assignments from my Spring 2026 independent study in computational linguistics. 

The coursework focuses on practical Natural Language Processing (NLP) methods, building foundational skills in text analysis and machine learning.

## Course Materials
* **Textbook:** [*Speech & Language Processing*](https://web.stanford.edu/~jurafsky/slp3/) by Dan Jurafsky & James H. Martin.
* **Assignments:** Originally created by [Rob Voigt for LING334](https://faculty.wcas.northwestern.edu/robvoigt/courses/2024_winter/ling334/) at Northwestern University. *(Special shoutout to Rob for sharing additional resources!)*

## Core Topics
* Language Modeling
* Text Classification
* Data Annotation
* Computational Semantics
* Machine Translation

## Final Project
This independent study culminated in a research project analyzing cross-cultural semantic drift using word2vec. 

View the final project and methodology here: **[What Bleeds Through: An Embedding-Based Case Study](https://github.com/carocla/what_bleeds_through)**

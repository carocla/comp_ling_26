# Computational Linguistics
This repository contains code and notes from my Spring 2026 independent study in computational linguistics. 
Textbook of reference is Speech & Language Processing by Jurafsky & Martin (https://web.stanford.edu/~jurafsky/slp3/)
Assignments created by Rob Voigt for LING334 (https://faculty.wcas.northwestern.edu/robvoigt/courses/2024_winter/ling334/).

Projects focus on practical NLP methods like language modeling, classification, annotation, semantics, and machine translation.
